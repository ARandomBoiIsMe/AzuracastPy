AzuracastPy: The Unofficial Python Wrapper for the Azuracast API
================================================================

AzuracastPy is a Python library that wraps the Azuracast API, allowing
for more straightforward use in Python.

This project is heavily "inspired" by `PRAW <https://github.com/praw-dev/praw/>`_.

.. note::

   This project is under active development.
